# SvelteKit Frontend - Development Rules & Reference

This directory contains the SvelteKit frontend for the **School Management Application**. The built assets are designed to be embedded in the Go backend.

## Constraints & Rules
- **CRITICAL**: Do not change or adjust any files outside this `frontend/` folder. All modifications must be limited to this directory.

## Commands
Run all commands inside the `frontend` directory:
- Development server: `bun run dev`
- Build for production: `bun run build`
- Type checking: `bun run check`
- Linting: `bun run lint`
- Formatting: `bun run format`
